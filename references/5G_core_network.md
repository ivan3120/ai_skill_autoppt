# 5G核心网基础知识

## 概述

5G核心网（5G Core Network）是5G系统的核心组成部分，负责提供用户面和控制面功能，实现移动性管理、会话管理、数据转发等功能。

## 架构组成

### 主要网元

| 网元 | 英文缩写 | 功能说明 |
|------|----------|----------|
| 接入和移动性管理功能 | AMF | 负责用户的移动性管理、注册管理等 |
| 会话管理功能 | SMF | 负责用户会话的建立、管理等 |
| 用户面功能 | UPF | 负责数据包的转发和路由 |
| 策略控制功能 | PCF | 负责策略规则的制定和执行 |
| 统一数据管理 | UDM | 负责用户数据的存储和管理 |
| 认证服务器功能 | AUSF | 负责用户身份认证 |
| 网络切片选择功能 | NSSF | 负责网络切片的选择 |

## 评估维度

### MT0: 网元高稳评估
评估网元的稳定性，包括：
- 设备健康状态
- 告警数量
- 性能指标

### MT1: 部署离散度评估
评估网元在不同站点的分布情况：
- 同城冗余部署
- 跨站点容灾

### MT2: 组网架构高可用评估
评估网络的冗余设计：
- 主备配置
- 负载均衡
- 故障切换能力

### MT3: 业务路由高可用评估
评估业务路由的可靠性：
- 路由备份
- 切换时间
- 数据一致性

### MT4: 网络容量-网元间均衡度评估
评估网元间的负载均衡：
- 流量分布
- 资源利用率

### MT5: 网元版本EOS评估
评估网元版本的EOS（End of Support）状态：
- 版本生命周期
- 升级计划

### MT6: 云平台版本EOS评估
评估云平台版本的EOS状态：
- OS版本
- 虚拟化平台版本
